Docker Container Images and Docker Hub!

What is a Docker Container Image?

A Docker container image is a lightweight, standalone, executable package of software that includes everything needed to run an application: code, runtime, system tools, system libraries and settings. A image is similar to a snapshot of a virtual machine and the advantage is once a image is created and working, it will work on every machine that runs it.

Docker Hub

Docker Hub is Docker's online registry were users have the ability to sign up and create, share, and pull images from public or prvate registries. Millions of users share there own images they have created or you can create your own test registry to create your own images!
