How to Launch and Stop Containers!

This page is to give a example of launching a container image and then stopping once everything needed from image is completed! Always good to stop containers that are running when you no longer need them!

Ubuntu Example! Launching a image!

In this exmaple we will be using a ubuntu image to create a ubuntu server container! I used the "docker run --name Ubuntu1 -it ubuntu bash" which will pull down the lastest ubuntu version, along with naming the container Ubuntu1 and it will automatically prompt with the command prompt for the ubuntu image.

Example of output:

MacBook-1:~ JL$ docker ps CONTAINER ID IMAGE COMMAND CREATED STATUS PORTS NAMES e926a7821bc4 ubuntu "bash" 5 minutes ago Up 3 seconds Ubuntu1 MacBook-1:~ JL$

How to stop the container when finished

Once you have finished using the container you can stop it using the "docker stop 'name of container'" command.

MacBook-1:~ JL$ docker stop Ubuntu1 Ubuntu1 MacBook-1:~ JL$ docker ps CONTAINER ID IMAGE COMMAND CREATED STATUS PORTS NAMES The image we just used is still saved so if you wanted to re launch the container then you just have to use the same run command!
